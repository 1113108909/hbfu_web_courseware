Angular

列出所有可生成文件:ng g --help
生成模块:ng g m
生成组件:ng g c
生成服务:ng g s
生成指令:ng g d
生成管道:ng g p
生成拦截器:ng g i

git

将目录修改后的文件加入暂存区: git add .
将文件修改同步到本地: git commit -m "desc"
将文件修改推送到远程: git push
拉取远程仓库代码: git pull
